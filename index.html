The intersection of Two linked-list

This code defines a class ListNode for creating a linked list data structure and a function getIntersectionNode find the intersection node of two linked lists. It also includes some example usage and code to display the linked lists in an HTML document.

The example usage creates two linked lists list1 and list2 with some nodes. It then finds the intersection node of the two lists using the getIntersectionNode function.

Example Usage:

const {ListNode, getIntersectionNode} = require('./index');
